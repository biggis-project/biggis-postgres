# BigGIS PostgreSQL
[![Build Status](https://travis-ci.org/biggis-project/biggis-postgres.svg?branch=master)]
Docker container for PostgreSQL

## Prerequisites
Docker Compose >= 1.9.0

## Deployment
```sh
docker-compose up -d
```

## Ports
- PostgreSQL is running on port `5432`
